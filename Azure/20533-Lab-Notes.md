## Module 7 Lab B Exercise 1

If an error occurs when using the `az aks create` command related to `<VM size>`, try one of the following sizes:

* `Standard_D2_v2`
* `Standard_D2_v3`
* `Standard_D1`
* `Standard_D4_v2`
